## Dev
- Install [`yarn`](https://yarnpkg.com/getting-started/install)
- Install dependencies
```sh
yarn install
```
- Install browser
```sh
yarn playwright install
```
- Test
```sh
yarn playwright test
```
- Runs the tests in a specific file.
```sh
yarn playwright test example
```
